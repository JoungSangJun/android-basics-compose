👨‍💻 InventoryApp
==================================
인벤토리 항목 목록을 표시하고 Room을 사용하여 앱 데이터베이스의 항목을 업데이트하고 수정하고 삭제할 수 있는 Inventory 앱입니다.

디자인 패턴은 MVVM을 활용하였고, 이전 CodeLab에서 배운 DI(의존성 삽입)을 사용하였습니다.

--------------
