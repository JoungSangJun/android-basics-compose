📚 Woof App
==================================
Introduction
------------
👉🏻 Woof App은 강아지 사진, 이름, 나이, 좋아하는 활동을 포함한 정보들의 리스트입니다.
이 앱은 Material Design를 사용하여 UI를 꾸몄으며 Animation을 추가해 생동감을 더하였습니다.
뿐만 아니라 Light Mode와 Dark Mode 둘 다 작동하도록 디자인했습니다.
