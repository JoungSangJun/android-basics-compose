📚 Day30Recipe App
==================================
Introduction
------------
👉🏻 Day30Recipe App은 매일 해서 먹을 요리를 알려주는 앱으로 이미지를 클릭하면 요리에 들어갈 재료를 알려줍니다.
이미지 클릭시 지료가 나올 때 Animation을 사용해 세련되게 만들었으며 Material Design을 통해 일관성 있는 앱을 만들었습니다.
