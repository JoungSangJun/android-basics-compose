👨‍💻 Mars Photos
==================================
MarsPhotos앱은 실제 화성이미지를 Grid 형식으로 화면에 보여줍니다.

화성이미지는 Retrofit을 사용해 웹 서버에서 가져오며 Coil을 사용해 이미지를 load 해줍니다.
